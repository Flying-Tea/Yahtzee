# Yahtzee

This is the game **Yahtzee** built as a short team project written in **C#** for a school project. Play against yourself, roll dice, and score points according to the Yahtzee rules.

# Features

- Roll 5 dice each turn
- Option to re-roll, hold and unhold
- Score categories: Aces, Full House, Yahtzee, etc.
- Console-based interface
- Score tracker
- Testing file

# Prerequisites

- [.NET SDK](https://dotnet.microsoft.com/download) (version 6.0 or later recommended)
- Visual Studio or Visual Studio Code

# Installation

Clone Repository
1. git clone https://github.com/Flying-Tea/Yahtzee.git
Navigate to project folder
2. cd Yahtzee (if outside file)
Build and run the project in console
3. dotnet run
